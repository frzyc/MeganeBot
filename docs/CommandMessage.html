Class: CommandMessage

CommandMessage(client, message, command, argString)

A class after the corresponding command from the message is parsed. This class primarily deals with setting up the environment to execute the command, parse the arguments for the command, and finally executing the command.

Does all the operations for execution of the CommandMessage#command.
